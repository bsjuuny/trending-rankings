# 🧠 Trending Rankings (관심사 분석 마인드맵) 개발자 가이드

안녕하세요! 이 프로젝트는 지금 인터넷 상에서 사람들이 무엇에 열광하는지(커뮤니티 이슈), 어떤 주식에 관심이 있는지(네이버/토스 검색어)를 한데 모아 **마인드맵**으로 보여주는 서비스입니다.

---

## 🏗 시스템 작동 원리 (The Concept)

이 서비스는 단순히 검색어만 가져오지 않고, **'얼마나 핫한가'**를 점수화합니다.

1.  **크롤링**: 여러 커뮤니티(디시, 루리웹 등)와 증권사 사이트에서 실시간 랭킹을 가져옵니다.
2.  **점수 계산 (Weighting)**: 조회수뿐만 아니라 "갑자기 언급이 늘어난 키워드"에 높은 점수를 줍니다.
3.  **마인드맵 배치**: `src/components/mindmap/` 로직이 데이터를 받아서, 점수가 높은 키워드를 큰 글씨로 화면 중앙에 배치합니다.

---

## 🛠 주요 명령어 및 작업 가이드

### 1. 데이터 수집 및 업데이트
- `npm run collect:all`: 모든 소스(커뮤니티, 주식, 글로벌)에서 데이터를 긁어옵니다. (수동 데이터 갱신 시 사용)
- `npm run generate`: 수집된 데이터를 바탕으로 텔레그램 발송용 요약 보고서(`summary.txt`)를 만듭니다.

### 2. 빌드 및 배포
- `npm run build:push`: 데이터를 수집하고 앱을 빌드한 뒤, Cafe24 서버로 자동 업로드하고 Git 커밋까지 해주는 **'한 방'** 명령어입니다.

---

## 💡 개발자를 위한 유지보수 팁

### 새로운 커뮤니티를 추가하고 싶다면?
1.  `scripts/` 폴더 내에 `collect-newsite.js`와 같은 이름으로 스크레이퍼를 만드세요.
2.  `package.json`의 `collect:all` 스크립트에 이 파일을 추가하면 시스템에 편입됩니다.

### 광고성 키워드를 거르고 싶다면?
- `lib/filter.js` 파일을 열어 금칙어 목록이나 정규 표현식을 추가하세요. 마인드맵이 깨끗해집니다.

---

## 🚨 문제 해결 (Troubleshooting)

**Q: 마인드맵 화면이 텅 비어 있어요.**
- A: `public/data/mindmap.json` 파일이 제대로 있는지, 혹은 파일 안에 유효한 데이터가 담겨 있는지 확인하세요. 수집 중 에러가 나서 파일이 비어 있을 수 있습니다.

**Q: 특정 사이트 수집이 안 돼요.**
- A: 해당 사이트가 **크롤링 방지**(CAPCHA 등)를 도입했거나, HTML 구조가 바뀌었을 가능성이 큽니다. `scripts/` 내의 해당 크롤러를 실행해 보고 로그를 확인하세요.

---

## 📁 주요 폴더 설명
- `scripts/`: 수집 엔진들의 집합소. 각 소스별로 코드가 분리되어 있어 유지보수가 쉽습니다.
- `src/components/mindmap/`: 이 프로젝트의 핵심인 마인드맵 시각화 로직이 담긴 곳입니다. (Tailwind CSS 4가 사용되었습니다.)
- `public/data/`: `mindmap.json` (웹 출력용)과 `summary.txt` (알림용)가 사는 곳입니다.
